## Why

用户对深色主题的需求日益增加，特别是在长时间使用的场景下，暗黑模式可以减少眼睛疲劳、提升用户体验。系统需要支持暗黑模式切换，同时保持后续扩展的灵活性。

## What Changes

- 实现全局暗黑模式切换功能，支持明/暗两种主题切换
- 提供 ThemeService 服务接口，支持通过 API 动态切换主题
- 在本地存储中持久化用户的主题偏好
- 预留动态切换接口，后续可通过按钮或其他组件调用

## Capabilities

### New Capabilities
- `dark-mode`: 提供暗黑模式切换服务，支持明/暗主题的动态切换、持久化存储及状态查询

### Modified Capabilities
- (无)

## Impact

- 前端主题样式系统
- 本地存储持久化
